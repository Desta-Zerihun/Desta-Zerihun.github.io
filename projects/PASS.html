PASS: The Pastoralists Animals Shepherd: (Resolve Conflicts and Handle Animal Rustling)

  • Tech Stack: Front-End-> HTML, CSS and JavaScript, Backend-> Java Servlet, Apache Cassandra
  • Publication URL: ieeexplore link

Abstract

    • This paper presents PASS, a new and innovative algorithmic based solution for resolving deadly conflicts that arise due to animal rustling and animal identification problems among the pastorals community in Ethiopia. The PASS solution proposes a tracking framework based on GPS enabled location sensors, the GSM/WCDMA wireless network and algorithms running in an edge clouds. Beside, PASS proposes an automatic digital identification mechanism that helps resolve conflicts during animals mix-up. This algorithmic based solution would totally relief the community from using the traditional identification mechanisms such as hot branding which are known to be cruel to animals. To communicate with the pastorals, PASS takes into consideration the low level literacy of the community as well as their use of low end mobile devices. Moreover, very low economy status, different environments of three kinds of livestocks, day and night variations of animal movements are all factored into the design of the algorithms. PASS has been evaluated using a simulated data and it works well to solve the mentioned problems with scale and reliability.
